# DesafioUiFlutter

Esse é o primeiro desafio de Flutter UI do grupo de estudos EPIC.

O desafio consiste em fazer uma tela inicial que contém:
 - Imagem Circular 
 - Nome 
 - Um texto
 - Um botão que ao pressionar mostrará a quantidade de clicks
